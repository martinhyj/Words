#### word in my way
* 用我自己的方式，记单词！！
* 结合书籍与与编程语言python，进行科学高效的记单词



#### 开发工具

	python3.x以及相关第三方库(sqlite3,tkinter，pandas,matplotlib)
	sqlite(sqlite manager)
